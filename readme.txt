notikumi.com es una agenda cultural. Llévate su contenido a tu blog.
Exposiciones, conciertos, obras de teatro, cine, deportes, infantil, festivales...

== Description ==

¿Por qué no creas una sección de eventos para tu blog? 
notikumi te ofrece sus ¡más de 100.000 eventos! para que tus lectores no se pierdan nada.  

Tras instalarte el plugin, podrás configurar qué contenido quieres mostrar. 
Puedes elegir de qué temática quieres que sean los eventos, de qué ciudad. 
 
¡Es gratis!

== Installation ==

Instalar el plugin es sencillo.

1. Busca notikumi en el buscador de plugins
1. Instálalo y actívalo
1. Ahora solo queda configurarlo desde: Ajustes -> notikumi
1, Y crear una página con [NTK] como contenido :)
